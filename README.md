# MercadolibreRails

Library to integrate MercadolibreApi into a Rails Project

## Usage

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'mercadolibre_rails'
```

And then execute:
```bash
$ bundle
$ rails db:migrate
```
